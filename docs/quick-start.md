# Quick Start

Rapidash doesn't need anything other than itself.

<p class="tip">
You can use it for both NodeJs and Browser environment.
</p>

## Install 
- npm
```bash
npm install rapidash
```

- yarn
```bash
yarn add rapidash
```


## Usage
```js
const r = require('rapidash');
console.log(r.max([1,2,3])); // 3
```
